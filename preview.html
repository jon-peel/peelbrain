Setting Up Org-Mode for Sleep Tracking

Posted: 2026-01-20

  • emacs
  • org-mode
  • quantified-self

I've been experimenting with using Emacs org-mode to track my sleep patterns. The combination of org-mode's property drawers and date-tree structure makes it surprisingly powerful for this kind of quantified-self work.

Why Org-Mode?

Most sleep tracking apps are black boxes. I wanted something I could query, export, and analyse with my own tools. Org-mode gives me structured data in plain text — I can write awk scripts, Python notebooks, or SQL queries against it.

The Basic Structure

Here's the property drawer I use for each night:

** 2026-01-19 Mon
   :PROPERTIES:
   :SLEEP:    23:00
   :WAKE:     07:30
   :QUALITY:  4
   :NOTES:    Too much coffee after 3pm
   :END:

The key bindings C-c C-x p and C-c C-x o make it quick to log entries without breaking flow.

Querying the Data

Using org's column view, you can get a tabular summary without leaving Emacs. For deeper analysis I export to CSV and run a short Python script to plot trends.

Column View Setup

Add this to the top of your file:

#+COLUMNS: %25ITEM %SLEEP %WAKE %QUALITY

Then C-c C-x C-c gives you a live spreadsheet view.
